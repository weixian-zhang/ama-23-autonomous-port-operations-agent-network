
## style guide

* hand sketch on paper cartoonish
* resolution: widescreen 16:9, 8k resolution. Text must be clearly visible and readable
* no grid lines

## tasks

* retain original diagram in infographics
* ensure all icons are visible
* all boxes remain intact, boxes can be colorful
* do not mix up line connectors from source to destination icons
* all text are clear and readable
* add cartoonish graphics to diagram to better explain
* Do not include title
