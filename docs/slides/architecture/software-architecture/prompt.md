
## style guide

* artist hand sketch, cartoonish on paper
* ensure infographics take up full space and is large and visible
* resolution: 8k resolution. Text must be clearly visible and readable
* no line border for all boxes
* no grid lines

## tasks

* retain original diagram in infographics
* all text are clear and readable
* free to add graphics to diagram to better explain but do not add duplicate graphics
* Do not include title
