
## style

A high-end, professional financial infographic dashboard set against a futuristic maritime port background at twilight. The central focus is a sleek, translucent holographic display showing rising bar charts, glowing data nodes, and 3D financial trend lines. The aesthetic is 'Industrial Futurism': deep navy blue and charcoal palette with accents of neon cyan and gold. In the background, blurred silhouettes of autonomous electric cranes and a massive container ship with glowing LED strips are visible. The lighting is cinematic with sharp focus on the data. Minimalist, clean UI/UX design, hyper-realistic, 8k resolution, sophisticated and corporate.

## task

* create infographics for port operation metrics and financial data.
* The Port Operation metric data and Financial Data are using a comparison between using mainstream terminal OS and Selesha, the next generation AI Agent powered terminal OS.
