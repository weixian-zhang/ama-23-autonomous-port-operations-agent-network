## Import export flow steps

Step 1 — Vessel Arrival
Vessel sends arrival notice into Port Community System
Shipping line sends Bay Plan via EDI
Port receives both arrival notice + Bay Plan and store in TOS
TOS assigns berth for vessel arrival

Step 2 — Vessel docks
Vessel berths
Quay Cranes start unloading

Step 3 — QC unloading
Multiple QCs lift containers from vessel
QCs place containers onto multiple AGVs

Step 4 — AGV transport
AGVs move containers from berth to yard

Step 5 — Yard stacking
Reach Stackers unloads container from AGV
Stack in assigned yard location

Step 6 — Import flow (reverse direction)
Haulage truck checks in at port gate

Step 7 — Truck enters port
Truck brings containers to yard or to berth for vessel loading
